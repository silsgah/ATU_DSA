## How to Get Started

### Step 1 — Create your private repository from this template

1. Click the green **"Use this template"** button at the top of this page.
2. Select **"Create a new repository"**.
3. Set the name to: `dsa-assignment-2-INDEXNUMBER`
   *(replace `INDEXNUMBER` with your actual student index number, e.g. `dsa-assignment-2-ATU2024001`)*
4. Set visibility to **Private**.
5. Click **"Create repository"**.

### Step 2 — Clone your repository and work locally

```bash
git clone https://github.com/atu-dsa-2025/dsa-assignment-2-INDEXNUMBER.git
cd dsa-assignment-2-INDEXNUMBER
```

### Step 3 — Implement your solutions

Open each `.py` file and complete the `TODO` sections.
**Do not change any function signatures or the test harness in the `__main__` block.**

### Step 4 — Test your work locally

Run each file individually to check your output against the expected values:

```bash
python3 part_a_complexity.py
python3 part_b_arrays.py
python3 part_c_linked_lists.py
python3 part_d_hash_bst.py
python3 part_e_bonus.py       # optional
```

### Step 5 — Commit and push your work

```bash
git add .
git commit -m "Complete Part A and Part B"
git push origin main
```

Push frequently. Your **last push before the deadline** is your submission.

### Step 6 — Transfer ownership to submit

When you are done, go to your repository:

**Settings → Danger Zone → Transfer ownership**

Transfer to the class organisation: **`atu-dsa-2025`**

> ⚠️ **Your submission timestamp is the GitHub server push time of your last commit.
> Local commit timestamps are not accepted as proof of on-time submission.**
