# Scenario: Smart University Digital File Management System

A university wants to build a **Digital Academic File Management System** to organize all academic resources in a hierarchical structure.

The university stores information in the following format:

```text
University
 ├── Faculty
 │    ├── Department
 │    │     ├── Level
 │    │     │     ├── Course
 │    │     │     │     ├── Lecture Notes
 │    │     │     │     └── Assignments


Design and implement a Python application that models the university’s academic storage system using a Tree Data Structure.

Your program should allow users to:

Create nodes in the hierarchy.
Add child nodes dynamically.
Display the complete tree structure.
Search for any node in the tree.
Calculate:
depth of a node,
height of the tree,
and number of nodes.
Display all leaf nodes.
Traverse the tree using:
Depth First Search (DFS)
Breadth First Search (BFS)
Print the path from the root node to a selected node.
