# University Grade Data Structures and Algorithms

Welcome to this 12-week comprehensive Data Structures and Algorithms curriculum. This course is designed to balance theoretical computer science rigor with practical, engineering-focused implementations. 

## Objectives

By the end of this course, you will be able to:
1. Analyze the time and space complexity of arbitrary algorithms using Asymptotic Analysis (Big-O).
2. Implement fundamental data structures from scratch (e.g., dynamic arrays, hash tables, linked lists, and graphs) and understand their memory layouts.
3. Apply standard algorithm patterns (Two Pointers, Sliding Window, DFS/BFS, Divide and Conquer, Dynamic Programming) to solve complex coding interview and systems engineering problems.
4. Make engineering trade-offs regarding which data structure best fits a particular set of constraints.

## Course Structure (12 Weeks)

- **Week 01:** Foundations & Algorithm Analysis
- **Week 02:** Arrays, Strings, and Pointers
- **Week 03:** Linked Lists
- **Week 04:** Stacks and Queues
- **Week 05:** Hash Tables & Sets
- **Week 06:** Trees and Basic Traversals
- **Week 07:** Heaps and Priority Queues
- **Week 08:** Graphs I (Fundamentals)
- **Week 09:** Graphs II (Advanced)
- **Week 10:** Divide and Conquer
- **Week 11:** Dynamic Programming I
- **Week 12:** Dynamic Programming II & Greedy Algorithms

## Format of the Material

Each week is enclosed in its own directory and contains three core files:
1. `README.md` - Lecture material, visualizations, and theoretical grounding.
2. `practice.py` - Skeleton code to practice implementations.
3. `solution.py` - Standard solutions to the exercises for verification and study.
4. `slides.md` - Presentation slides for quick review of the week's topics.

*Language:* Python (Standard) - Python is widely used in coding interviews due to its concise syntax, and it comes with excellent standard libraries. However, it's highly encouraged to think about memory management as if you were working in C++ or Java during these exercises.
